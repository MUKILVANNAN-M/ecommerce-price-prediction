# 📊 E-Commerce Price Prediction & Review Intelligence System

## 📌 Overview
A machine learning system that predicts product prices and analyzes customer reviews using NLP and AI techniques.

---

## 🚀 Features
- Price prediction using Random Forest
- Sentiment analysis
- Spam detection
- Review summarization
- Topic modelling

---

## 🛠 Technologies
- Python
- Scikit-learn
- PyTorch
- Streamlit
- Google Gemini API

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Add API key:
Replace YOUR_API_KEY in code

3. Run:
streamlit run app.py
